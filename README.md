# SQL_PROJECT
Pizza Sales Analysis Project  
This project focuses on analyzing retail and pizza sales data using MySQL to extract meaningful business insights. 
The dataset includes customer details, order information, product categories, pricing, and sales transactions. 
SQL queries were used for data cleaning, aggregation, and trend analysis.

Key objectives of the project:
1.Clean the dataset by identifying and handling NULL values.
2.Analyze sales performance by category, customer demographics, and product types.
3.Determine top-selling products and highest revenue contributors.
4.Study order patterns by date and hour to understand peak demand times.
5.Calculate metrics such as total sales, quantity sold, and average orders per day.

Tools & Technologies:
1.MySQL for querying and data analysis
2.Excel for dataset storage and basic preprocessing
3.GitHub for version control and project documentation
